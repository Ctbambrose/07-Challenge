# 07-Challenge-README-Generator

## Description

READMEs are very important to have as a developer, and having a quick and effective way of generating them would make the process faster and easier to accomplish.

## Installation

No steps are really needed to install.

## Usage

To run the application all that you need to do is run the following command from the terminal:
node index.js
